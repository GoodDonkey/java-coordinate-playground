
## CoordinateValue [class]
- 1개의 값을 가진다.
- 1개의 값 범위를 가진다.
- 값을 검증을 수행한다.
- 값이 같은지 비교가능하다

## ValueRange [interface]
- 값 범위를 검증한다.

### LessThanOrEquals24ValueRange [class]
- 값이 24 이하인지 검증한다.

## TwoDimensionalCoordinate [class]
- 2개의 값으로 좌표를 표현한다.
- 좌표값이 같은지 비교가능하다.

## AbstractTwoDimensionalShape [abstract class]
- 2개 이상의 좌표값으로 2차원 도형을 표현한다.
- 각 좌표는 같아서는 안된다.

## Line [class]
- 2개의 좌표값을 가지는 선분이다.

## Rectangle [class]
- 4개의 좌표값을 가지는 직사각형이다.

## Triangle [class]
- 3개의 좌표값을 가지는 삼각형이다.
